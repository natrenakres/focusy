# Product Requirements

## Must Have
- A user can create a project
- A user can create a task 
- A user can add task to project
- A user can add milestone to project
- A user can start and end a focus session for task
- A user can edit settings for session, pomodoro timer
- Session duration is persisted
- User can see a daily summary

## Should Have

- User can see a weekly summary
- User can see project progress


## Could Have
- Basic AI-generated suggestions (post-V1)
