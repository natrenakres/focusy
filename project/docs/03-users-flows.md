# User Flows

## Core Flow
User opens app
→ Logs in
→ Starts focus session
-> Start pomodoro timer for session
→ Ends or aborts session
→ Views summary

## Project Flow
User opens app
-> Logs in
-> Create Project
-> Add a milestones to project
-> Add a task to project
-> Views a progress of project


## Edge Flow
User closes browser during session
→ Session is safely recoverable
