# Scope Definition

## In Scope (V1)
- All data should be located in locale 
- User authentication (login / logout)
- Create a project
- Create a task for a project
- Start, pause, end focus sessions for e task
- Track session duration
- Daily and weekly summaries


## Explicitly Out of Scope (V1)
- Team or collaboration features
- Gamification (points, streaks, badges)
- Mobile applications
- Advanced analytics or exports
- Individuel Finance
- AI support for individiel finances
